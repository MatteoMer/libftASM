# libftASM
An ASM mini-library with some functions

#### My own implentation in ASM (macho64) of
- bzero
- strcat
- isalpha
- isdigit
- isalnum
- isascii
- isprint
- toupper
- tolower
- puts

#### These function needed to be wrote with Instruction Repeat String Operations
- strlen
- memset
- memcpy
- strdup

#### A little implentation of `cat` in ASM
- ft_cat
